# Какво е НСИ Асистент?

**НСИ Асистент** е браузърна добавка, която автоматизира попълването на годишните финансови отчети (ГФО) в информационната система „Бизнес Статистика" на НСИ.

## Основни възможности

- **Автоматично попълване** на ГФО справките директно в [isbs.nsi.bg](https://isbs.nsi.bg)
- **Работа с Excel файлове** - използва данни от вашите съществуващи файлове
- **Универсална съвместимост** - работи със съществуващи файлови структури без нужда от преформатиране

## Ключови характеристики

| Характеристика | Описание |
|----------------|----------|
| **Местна обработка** | Данните остават на вашето устройство - не се изпращат на външни сървъри |
| **Поддържани браузъри** | Google Chrome и Microsoft Edge |
| **Операционни системи** | Windows (включително Windows 7), macOS, Linux |
| **Сигурност** | Не изисква достъп до акаунти на Бизнес Статистика |

## Основни предимства

- **Спестяване на време** - автоматизира ръчното прехвърляне на данни
- **По-малко грешки** - елиминира човешкия фактор при въвеждане
- **Ефективност** - позволява обработка на множество фирми

## Как да започнете?

1. **[Регистрация](https://nsiassistant.bg)** - създайте акаунт в платформата
2. **[Инсталиране](/docs/install/install_extension)** - добавете разширението към браузъра
3. **[Конфигуриране](/docs/guide/configuration)** - адаптирайте системата към вашите Excel файлове
4. **[Автоматично попълване](/docs/guide/auto-fill)** - стартирайте процеса

За пълно описание на всички функционалности, вижте **[Ръководството](/docs/guide)**.
