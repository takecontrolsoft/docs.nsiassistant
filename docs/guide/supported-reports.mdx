# Поддържани справки

НСИ Асистент поддържа всички ключови отчети в системата "Бизнес статистика".

## Списък на поддържаните справки

| Код | Наименование |
|-----|--------------|
| **ПРЕДПР** | Справка за предприятието |
| **ГРУПАПР** | Справка за група предприятия |
| **БАЛАНС** | Счетоводен баланс |
| **ОПР** | Отчет за приходите и разходите |
| **КАПИТАЛ** | Отчет за собствения капитал |
| **ПАРПОТОЦИ** | Отчет за паричните потоци по прекия метод |
| **ДМА** | Справка за нетекущите (дълготрайните) активи |
| **МИКРОДА** | Справка за нетекущите (дълготрайните) активи на микропредприятията |
| **ТАКСИКОМ** | Справка за таксите и комисионите |
| **ЗАЕМИ** | Справка за заемите и начислените лихви по институционални сектори и подсектори |
| **РДМА** | Справка за разходите за придобиване на дълготрайни материални активи |
| **ЕКОТАКСИ** | Справка за данъците и таксите |
| **СУБСИДИИ** | Справка за начислените субсидии и инвестиционни помощи и извършените разходи |
| **ВЗЕМАНИЯ** | Справка за вземанията и задълженията, разпределени по институционални сектори |
| **ТРУД** | Отчет за заетите лица, средствата за работна заплата и други разходи за труд |
| **СТР** | Справка за приходите на строителните предприятия по видове строителство |
| **НЕТНИ** | Справка за приходите от продажби на предприятията, предоставящи бизнес услуги по видове продукти |
| **РСМАТ** | Справка за разходите на суровини и материали |
| **ГОРИВА** | Справка за потреблението на горива и енергия |

:::tip Достъп до справките
Всички справки са достъпни от менюто **[Справки](https://nsiassistant.bg/views)** в уеб платформата.
:::
